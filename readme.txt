Portfolio-HTML
    This is a simple project that showcases my skills in HTML and CSS. 
    I created a personal portfolio that displays some of my projects.

Features
    Responsive design that adapts to different screen sizes and devices.
    Custom fonts and icons from Google Fonts and Font Awesome.
    Animated cards, forms, images using CSS.
    Contact form that sends messages to my email using Formspree.
